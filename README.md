## Overview

Smartflow-CRM is a flexible and powerful customer relationship management software that integrates advanced AI features to manage client interactions efficiently.

## AI Features

- Intelligent Lead Scoring
- Predictive Analytics for Sales Forecasting
- Automated Customer Support through AI Chatbots

## Tech Stack

- **Frontend**: React, Redux
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Deployment**: Docker, AWS

## Installation Guide

1. Clone the repository
   ```bash
   git clone https://github.com/Shivam09xc/Smartflow-CRM.git
   ```
2. Navigate into the project directory
   ```bash
   cd Smartflow-CRM
   ```
3. Install dependencies
   ```bash
   npm install
   ```
4. Start the development server
   ```bash
   npm start
   ```
